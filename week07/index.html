    Chapter 13: AJAX

    AJAX, or more accurate AJAJ because in most cases we are returning Json objects and not XLM objects, has pushed the UI and UX of web pages because it allows us to load and update a web page without a full reload of the page. you can update parts of a page using API's. These API will fetch data from a server and bring back a response for our webpage to use. Since API's are a major part of web development now this topic is very complex and growing with complexity. I am a little worried about keeping up with API standards and methods. JQuery for example is a great library to use but there with surely be more like it in the future.
